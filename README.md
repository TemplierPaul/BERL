# BERL
## Benchmarking Evolutionary Reinforcement Learning

Framework to implement and compare evolutionary RL algorithms on multiple environments.

### Implemented:

Evolution with Evolutionary Strategies:
- Canonical ES
- SNES
- CMAES

Environments:
- Gym
- Minatar
- Atari
- Swingup
- Custom MountainCar
